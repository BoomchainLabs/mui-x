# MUI X Data Grid Premium - Aggregation and Row Grouping

A client-side React app demonstrating aggregation and row grouping with the MUI X Data Grid Premium.

For a step-by-step guide, see the [Aggregation and row grouping tutorial](https://mui.com/x/react-data-grid/tutorials/aggregation-row-grouping/) in the docs.

## Setup

```bash
pnpm install
```

## Run

```bash
pnpm dev
```

App runs on `http://localhost:5173`.
